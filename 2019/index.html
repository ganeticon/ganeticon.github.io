GanetiCon 2019 Umeå/Sweden

The Ganeti users and developers summit

Schedule

GanetiCon 2019 is scheduled for the 18th and 19th of June (2019-06-18 - 2019-06-19).

Call for Participation / Registration / Lectures

If you like to attend the GanetiCon 2019, please register on NeIC's event organization tool, so we can better plan the event. The participation list is publicly available.

Please help making GanetiCon a valuable event for everyone by sharing your knowledge and experience. If you like to give a lecture you can submit an abstract.

Practical Information

You can find helpful information like Venue, Accommodation and how to get there here.

Sponsors

GanetiCon 2019 will be hosted by Umeå University/Sweden on behalf of Mattias Wadenstein.
